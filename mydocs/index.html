    简介¶

    这是一个非常粗糙的在线公开笔记本，主要用来记录和备份平日学到的东西。

    Repo 地址在这里： Notebook。

    本笔记本使用 MkDocs 构建，采用 Material for MkDocs 主题。中文字体是 霞鹜文楷屏幕阅读版，英文字体是 JetBrains Mono（没想到 jbmono 的网页版本也支持连字 ）。

    欢迎 Fork 和 Star，也欢迎各位大佬提出 Issue 和 PR！

    在线阅读¶

    本项目托管在 GitHub Pages 上，可以通过 https://l6y6j6.github.io/mydocs/ 在线阅读。

    本地构建部署¶

    如果你想在本地构建并部署这个项目，可以按照以下步骤操作：

    首先请确保你已经安装了 Python 和 pip。

    1. 克隆项目到本地并切换到项目目录：
    1
2
    git clone https://github.com/L6y6j6/L6y6j6.github.io.git
cd L6y6j6.github.io
    1. 创建并激活虚拟环境：
    1
2
3
4
    python3 -m venv . # 在当前目录创建虚拟环境
source bin/activate # 激活虚拟环境（Linux，macOS）
# 或者
.\Scripts\activate # 激活虚拟环境（Windows）
    1. 安装依赖：
    1
    pip install -r requirements.txt
    1. 构建并启动项目：
    1
    mkdocs serve
    1. 在浏览器中预览

    打开浏览器，输入 http://127.0.0.1:8000/mydocs/ 即可预览。

    我的装备¶

    电脑¶

    • 惠普暗影精灵 9

      • 大名鼎鼎的 黑屏精灵

      • 伪装成台式机的游戏本，目前的主力机器。

      • Win 11 + Ubuntu + Arch 三系统

      • 性能强悍，风扇狂转，发热严重

      • i9-13900HX + RTX 4060

      • 32GB RAM / 1TB + 2TB SSD

      • 开发，娱乐，办公一把梭

    • MacBook Pro 2020

      • 最后一代 Intel 芯片的 MBP

      • 四核 Intel Core i5 + Intel 集显

      • 16GB RAM / 512GB SSD

      • 发热有些严重，续航一般

      • 屏幕，键盘，触控板体验一流

      • 轻便易携，日常开发完全够用

    • ThinkPad L480

      • 7 年前发售的经典商务本，开机依然耐用，做工依然耐操，外观依然耐看，性能依然稳定

      • Thinkpad + Win 10 + Arch 双系统 === 生产力利器 + 背包必备

      • 机身触感好，键盘手感佳，小红点很方便

      • 8GB RAM / 256GB + 1TB SSD

      • i7-8550U + Radeon 530

    • Thinkpad X240

      • 11 年前发售的“轻薄便携”本，配置在今天看来很丐，但配上 SSD + Arch Linux 依然流畅

      • Thinkpad 的做工还是相当好的，轻量使用还是很舒服的

      • 目前拿来当作测试机，以后可能会拿来跑些轻量服务

      • 双核 Intel Core i3-4010U + Intel 集显

      • 8GB RAM / 128GB + 320GB SSD

    • Thinkpad X201

      • 上古时代的笔记本电脑，配置已是丐中丐，Win 7 / Arch Linux 还不至于卡顿

      • Celeron U3400 + Intel 集显

      • 4GB RAM / 128GB SSD

      • 目前有如下处理方案：

        • 改造成小型 NAS 使用

        • 装上 Linux / BSD，做服务器使用

    • 惠普暗影精灵 2

      • 基本介绍

      • CPU + GPU

      • GB RAM / GB SSD

      • 使用概况

      • 屏幕，键盘等个性化体验

      • 目前主要用途

    • 机械革命笔记本

      • 基本介绍

      • CPU + GPU

      • GB RAM / GB SSD

      • 使用概况

      • 屏幕，键盘等个性化体验

      • 目前主要用途

    外设¶

    • Lenovo 27 英寸 显示器

      • 宿舍桌面上刚好能放下 27 英寸的显示器

      • 2K 分辨率，120Hz 刷新率，不可升降旋转

      • 目前横向放置，充当 HP 暗影精灵 9 的主扩展屏幕

      • RTX 4060 Laptop 能带动 2K 显示器，效果还不错

      • 大屏在分屏，观影，游戏时优势明显

    • AOC 21.5 英寸 显示器

      • 1080P 分辨率，75Hz 刷新率，可升降旋转

      • 宿舍桌面侧边放置，空间有些紧张

      • 目前竖向放置，充当 HP 暗影精灵 9 的第二扩展屏幕

      • 1080P + Windows 糟糕的字体渲染其实对长期伏案者不太友好，不过用来看看文档和代码也够了

    • 希捷移动硬盘 5TB

      • 赛博仓鼠必备（有 NAS 的当我没说）

      • 机械硬盘，传输速度一般，但容量大，价格便宜

      • 主要用途：

        • 备份，存储一些大文件；

        • 构建自己的影视音乐库；

        • PT 做种。

    • 拓展坞 x2

    • 机械键盘 x3

    • 蓝牙耳机

    • 鼠标 x2

    移动设备¶

    • 小米 13

      • 人生的第一台智能手机，高考后购入

      • 小屏旗舰，握感良好，性能不错

      • 骁龙 8 Gen 2 芯片，23年安卓顶配？

      • 许多功能是 IOS 没有的，或许是因为我用惯了 MIUI 的原因吧

      • 12GB RAM / 256GB ROM，内存够用，存储勉强够用（512GB 还是靠谱些哈哈哈）

      • 玩游戏时略微发热，总体续航不错

    • iPad Air 5

    • 小米手环 8
